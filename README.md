# ❌⭕ Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🎮 How to Play
- Two players take turns: X and O
- Click on a box to make your move
- First player to get 3 in a row wins
- If all boxes are filled, it's a draw

## 🛠️ Built With
- HTML
- CSS
- JavaScript

## 💡 About
This is a beginner project made to practice DOM manipulation and game logic in JavaScript.

Have fun playing! 🎉
